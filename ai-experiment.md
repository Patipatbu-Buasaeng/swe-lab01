### ขั้นตอนที่ 1: อธิบายโค้ดชุดใหม่ (โจทย์: ระบบคำนวณราคาสินค้าพร้อมส่วนลด)
สมมติว่านี่คือโค้ดที่ผู้สอนแจกมาให้ครับ:

```python
def calculate_price(amount, price_per_unit):
    total = amount * price_per_unit
    if total > 1000:
        discount = total * 0.1
    else:
        discount = 0
    final_price = total - discount
    return final_price

print(calculate_price(5, 300))
```

**คำอธิบายทีละบรรทัด (ฉบับคนไม่เคยเขียนโปรแกรม):**
1.  **`def calculate_price(amount, price_per_unit):`** – สร้างเครื่องคำนวณชื่อ `calculate_price` โดยต้องใส่ "จำนวนสินค้า" และ "ราคาต่อชิ้น" เข้าไป
2.  **`total = amount * price_per_unit`** – เอาจำนวนสินค้ามาคูณกับราคาต่อชิ้น เพื่อหา "ราคาก่อนหักส่วนลด"
3.  **`if total > 1000:`** – ตรวจสอบเงื่อนไขว่า ถ้าราคารวมมัน "มากกว่า 1,000 บาท" ใช่หรือไม่?
4.  **`discount = total * 0.1`** – ถ้าใช่ (มากกว่า 1,000) ให้คำนวณส่วนลด 10% (เอา 0.1 ไปคูณ)
5.  **`else:`** – แต่ถ้า "ไม่ใช่" (ราคาไม่ถึง 1,000)
6.  **`discount = 0`** – ก็จะไม่ให้ส่วนลดเลย (ส่วนลดเป็น 0)
7.  **`final_price = total - discount`** – เอาราคารวมตั้งต้น ลบด้วยส่วนลดที่คำนวณได้ เพื่อหา "ราคาสุดทธิ"
8.  **`return final_price`** – ส่งค่าราคาสุทธิออกไปบอกคนใช้งาน
9.  **`print(calculate_price(5, 300))`** – ลองสั่งงานเครื่อง โดยใส่จำนวน 5 ชิ้น ชิ้นละ 300 บาท (รวม 1,500 บาท ซึ่งเกินพัน) แล้วแสดงผลออกมาบนหน้าจอ

> ** AI อธิบายได้ถูกต้องตามหลักการทำงานของโปรแกรม

### ขั้นตอนที่ 2: เขียนฟังก์ชันใหม่ 
ผมจะเขียนฟังก์ชัน **"ค้นหาคะแนนที่สูงที่สุดในห้องเรียน"** 

```python
def find_max_score(score_list):
    if not score_list:
        return "ไม่มีข้อมูลคะแนน"
    
    highest = score_list[0]  # สมมติว่าคนแรกคะแนนเยอะสุดก่อน
    for s in score_list:
        if s > highest:      # ถ้าเจอคนใหม่ที่คะแนนเยอะกว่า
            highest = s      # ให้เปลี่ยนเอาคะแนนคนนั้นมาเป็นคะแนนสูงสุดแทน
            
    return highest

# --- ส่วนของ Test Case (กด Run ได้เลย) ---
test_scores = [45, 88, 92, 70, 55, 99, 82]
result = find_max_score(test_scores)

print(f"รายการคะแนนทั้งหมด: {test_scores}")
print(f"คะแนนที่สูงที่สุดคือ: {result}")
```
